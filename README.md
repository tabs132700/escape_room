# PHP Escape Room Platform

## Professional Title

Database-driven escape room web application built with PHP, JavaScript, and MySQL.

## Overview

This project is an interactive escape room web application where players create an account, log in, solve puzzles, track progress, and compare scores on a leaderboard. The application also includes an admin dashboard for managing players, sessions, and puzzle visibility.

## Features

- User registration and login
- Session-based player progress
- Database-driven questions and answers
- Puzzle completion tracking
- Score and hint system
- Leaderboard with ranked results
- Admin dashboard for user and puzzle management
- Win and lose flow with stored session results

## Tech Stack

- PHP
- JavaScript
- HTML5
- CSS3
- MySQL
- XAMPP or WAMP

## What I Learned

- Building a multi-page PHP application with authentication
- Managing user sessions and storing gameplay state
- Connecting PHP to MySQL for a database-driven application
- Tracking player progress, scores, and statistics
- Creating admin functionality for managing data and users

## Installation and Use

1. Clone the repository.
2. Create a local MySQL database named `escape_room`.
3. Add the required tables for `users`, `questions`, `game_sessions`, and `game_stats`.
4. Update the database connection values in the PHP files if your local setup is different.
5. Run the project in a local PHP environment such as XAMPP or WAMP.
6. Open `index.php` in your browser and create a user account to start playing.
